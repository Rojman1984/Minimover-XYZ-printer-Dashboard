    Troubleshooting Guide

    Printer Not Responding

    1. Check Connection: Verify USB cable is connected
    2. Power Cycle Printer: Turn printer off, wait 10 seconds, turn back on
    3. Restart Dashboard: 
      cd ~/Minimover-XYZ-printer-Dashboard
sudo pkill -f "node.*server.js"
npm start
    4. Check Serial Port: 
      ls -la /dev/ttyACM* /dev/ttyUSB*

    Camera Not Working

    1. Use Reconnect Button: Click the ⟳ button next to camera dropdown
    2. Check Camera Device: 
      ls -la /dev/videousb*
    3. Restart Camera Service: 
      sudo systemctl restart mjpg-streamer
sudo systemctl status mjpg-streamer
    4. Check Camera Connection: Replug USB camera, wait 5 seconds

    Upload/Print Failures

    1. File Format: Ensure file is .gcode or .3w format
    2. Printer Ready: Wait for "Connected" status before uploading
    3. Clear Jam: If printer frozen, power cycle and restart dashboard

    View System Logs

    # Dashboard logs (if running as service)
sudo journalctl -u minimover-dashboard -n 50

# Camera service logs
sudo journalctl -u mjpg-streamer -n 30

# System messages
sudo dmesg | tail -30

    Emergency Reset

    If dashboard becomes completely unresponsive:

    # Kill all node processes
sudo pkill node

# Restart services
sudo systemctl restart mjpg-streamer

# Start dashboard again
cd ~/Minimover-XYZ-printer-Dashboard
npm start
